# TrustHome

Mobile real estate and property management app — listings, property details, virtual tours, and agent communication.

**Live:** [trusthome.tlid.io](https://trusthome.tlid.io)

## Stack

| Layer | Tech |
|---|---|
| Framework | Expo (React Native) |
| Navigation | Expo Router |
| Backend | Trust Layer API (dwtl.io) |
| Auth | Trust Layer SSO |

## Structure

```
trusthome/
├── app/              # Expo Router screens
├── components/       # Shared UI components
├── constants/        # App configuration
└── assets/           # Images and fonts
```

## Development

```bash
npm install
npx expo start
```
